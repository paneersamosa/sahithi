COACHART AUTOMATED SYSTEMS

NOVEMBER 2023 - JUNE 2024

Project Scope: Part of Nova, Tech for Good, 2023-24

CoachArt is a non-profit organization that connects kids affected by serious illnesses with volunteers from all over the world to allow kids to learn an art/skill. These volunteers can request supplies that are necessary for a lesson or request reimbursement after purchasing.

However, both of these forms were set up as lengthy Google Forms, requiring a lot of manual labor from CoachArt's end in regards to inputting the data into a separate financial management platform.

My role in this project was to redesign the content from the unstyled Google Forms into an intuitive, visually pleasing webpage that could be integrated into their website. This would allow for the devs to create a software that could pull data from the form and generate automatic pdf invoices that could be uploaded onto the financial management platform.
